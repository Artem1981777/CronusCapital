## The Three Oracles

| Agent | Role | Output |
|-------|------|--------|
| 🔭 Scout | Market Intelligence | Signals with confidence scores |
| ⚖️ Analyst | Expected Value Engine | +EV bets, position sizing |

### Reasoning Trace
Every agent logs its thinking in real-time via collapsible Reasoning Trace panel.
